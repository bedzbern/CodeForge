# CodeForge Student

VS Code extension that connects student workstations to the CodeForge AI assistant.

## Features

- **Ask Questions** — Type a question about your code and get an AI-guided response.
- **Grab Selected Code** — Pull selected text from your editor into the question form.
- **Socratic Levels** — Responses adapt to the CodeForge teaching level system.

## Configuration

| Setting | Default | Description |
|---|---|---|
| `codeforge.serverUrl` | `http://192.168.1.1:8000` | URL of the CodeForge server |

## Usage

1. Open the CodeForge sidebar from the activity bar.
2. Type your question.
3. Optionally click **Grab Selected Code** to attach your selection.
4. Click **Ask CodeForge** and read the response.
